# Daily English
영단어 맞추기 어플

### 주요기능

1. 영단어 저장기능 (중복 x)
2. 완전 랜덤하게 출제되는 퀴즈
3. 오늘의 통계 확인가능  
  3-1. 오늘 총 틀린 단어들   
  3-2. 오늘 총 몇번 퀴즈 했지?  
  3-3. 오늘 평균 맞춘 개수    

### Data Base

SQLite 사용

1. word.db : 단어, 의미 저장

2. board.db : 날짜(yyyy-mm-dd), 틀린단어, 맞춘개수 저장

***

#### 1. 영단어 저장기능

영단어와 뜻을 입력받고 기존의 단어-뜻 쌍과 비교 한 뒤

없다면 word.db에 저장 후 리스트 뷰 업데이트

이미 존재하는 단어거나 혹은 아무런 입력을 하지 않았다면 

-> "단어가 이미 존재합니다." 라는 알림 발생
